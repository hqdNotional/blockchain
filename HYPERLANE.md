# HYPERLANE

### 1. Hyperlane CLI
#### 1. Installation
Npm:
```
npm install -g @hyperlane-xyz/cli
```

Npx:
```
npx @hyperlane-xyz/cli
```

Yarn dlx:
```
yarn dlx @hyperlane-xyz/cli
```

Build from source:
```
git clone https://github.com/hyperlane-xyz/hyperlane-monorepo.git
cd hyperlane-monorepo
yarn install && yarn build
cd typescript/cli
yarn hyperlane
```
